
### Courses routes
#### Create a course:
`http://localhost:5000/courses/create`
```json
{
"title": "Advanced programming",
"shortDescription": "Advanced progrmming course",
"description": "Advanced programming course about code",
"price": 0,
"organisationId": 1,
"createdBy": 5
}
```
